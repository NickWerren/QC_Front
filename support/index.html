If you like this project, then the best way you can support it is by interacting with the work hosted here and subscribing to The Quantized Crow Newsletter. The newsletter is a collection of irregular creative releases that can't make it as articles for whatever reason.

The work invested into this project is not paid. Quantized Crow was created, and is maintained, through significant investments of hard work and lots of time, but no money. If you want to support the project then you can buy me a coffee through KoFi.
